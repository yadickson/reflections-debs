# reflections-debs
Build Debian Package for reflections

**Build dependencies**

- debhelper (>= 9)
- cdbs
- default-jdk
- maven-debian-helper (>= 1.5)
- libguava-java
- libjavassist-java
- libdom4j-java
- libgoogle-gson-java
- libcommons-vfs-java

**Tested**

- Debian wheezy
- Debian jessie
